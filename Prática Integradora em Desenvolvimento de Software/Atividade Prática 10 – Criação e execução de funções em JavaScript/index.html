Função em JavaScript

O código para a função que valida o nome e sobrenome é:

          
            function obterNomeCompleto(nome, sobrenome) {
              if (nome === "" && sobrenome === "") {
                return "Nome e sobrenome inválidos!";
              } else if (nome === "") {
                return "Nome inválido!";
              } else if (sobrenome === "") {
                return "Sobrenome inválido!";
              } else {
                return nome + " " + sobrenome;
              }
            }

Guia de Execução

Passo a Passo

Objetivo: Você irá rodar um arquivo HTML que utiliza JavaScript para validar e exibir o nome completo de um cliente. O código verifica se o nome e o sobrenome estão preenchidos corretamente e, se não, retorna uma mensagem de erro.

Passo a Passo

  1. Baixar ou Obter o Arquivo

    Se você já possui o arquivo index.html (conforme descrito no exercício), basta passar para o próximo passo. Caso contrário, peça para alguém te fornecer o arquivo ou faça o download de um arquivo pronto que contenha o código mencionado anteriormente.

  2. Abrir o Arquivo no Navegador

    Localize o arquivo index.html em sua pasta ou no local onde ele foi salvo. Clique duas vezes sobre o arquivo para abrir automaticamente no seu navegador de internet preferido (Google Chrome, Firefox, Microsoft Edge, etc.). O navegador irá carregar a página e você verá um título na tela, algo como "Exemplo de Função JavaScript".

  3. Verificando o Resultado no Console do Navegador

    Para visualizar o resultado do código JavaScript, abra o console de desenvolvedor do seu navegador:

    • Google Chrome: Pressione Ctrl + Shift + I ou F12, e depois clique na aba "Console".
    • Firefox: Pressione Ctrl + Shift + I ou F12, e depois clique na aba "Console".
    • Microsoft Edge: Pressione Ctrl + Shift + I ou F12, e depois clique na aba "Console".

    Depois de abrir o console, você verá mensagens sendo exibidas. O código já contém alguns testes de validação. As mensagens esperadas são:

    • "João Silva" (caso o nome e sobrenome sejam válidos).
    • "Nome inválido!" (caso o nome esteja vazio).
    • "Sobrenome inválido!" (caso o sobrenome esteja vazio).
  4. Testando com Variações

    Caso você queira testar outros cenários:

    • Nome inválido: No arquivo HTML, altere o nome para uma string vazia (""), como no código abaixo: 
      console.log(obterNomeCompleto("", "Silva"));
    • Sobrenome inválido: No arquivo HTML, altere o sobrenome para uma string vazia (""), como no código abaixo: 
      console.log(obterNomeCompleto("João", ""));

    Salve o arquivo depois das mudanças e atualize a página no navegador (pressione Ctrl + R ou F5 para atualizar a página). O console agora exibirá o resultado da nova execução.

Conclusão

Agora você é capaz de validar entradas de nome e sobrenome e retorná-las ou emitir mensagens de erro caso os dados sejam inválidos. Isso é fundamental para garantir a integridade dos dados fornecidos pelo usuário.
